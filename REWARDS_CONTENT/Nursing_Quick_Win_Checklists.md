## CHECKLIST 1: NMC Revalidation - 30 Days to Submission

**Your Revalidation Deadline is Approaching!**

Use this checklist in the final month before submission.

### ‚úÖ WEEK 1: Evidence Collection

**Practice Hours (450 minimum over 3 years)**
- [ ] Collect all payslips from last 3 years
- [ ] Check NHS ESR printout (if applicable)
- [ ] Document bank/agency shifts
- [ ] Calculate total hours (should exceed 450)
- [ ] Create spreadsheet: Date | Location | Hours | Running Total
- [ ] Save as PDF for evidence

**CPD Hours (35 minimum over 3 years)**
- [ ] Gather ALL certificates (mandatory training, study days, courses)
- [ ] Check RCN/online learning records
- [ ] Document informal learning (ward teaching, journal reading)
- [ ] Create CPD log spreadsheet
- [ ] Ensure you have 35+ hours
- [ ] Save as PDF for evidence

**Professional Indemnity**
- [ ] Locate current indemnity certificate (RCN/Unison/standalone)
- [ ] Check it covers your practice area
- [ ] Verify expiry date is AFTER revalidation date
- [ ] Save copy as PDF

## CHECKLIST 3: Shift Handover Excellence - SBAR Template

**Give handover like a pro every time!**

### The SBAR Framework

**S - Situation**
‚ñ° Patient name, age, location (bed number)
‚ñ° Presenting complaint/reason for admission
‚ñ° Admission date
‚ñ° Medical team

**B - Background**
‚ñ° Relevant medical history
‚ñ° Allergies
‚ñ° Baseline observations
‚ñ° Current treatment plan
‚ñ° Mobility status
‚ñ° Nutritional status

**A - Assessment**
‚ñ° Current condition
‚ñ° Latest observations (time + values)
‚ñ° Pain score
‚ñ° Skin integrity
‚ñ° Mood/mental state
‚ñ° Any concerns/red flags

**R - Recommendations**
‚ñ° Outstanding tasks (bloods due, medications pending)
‚ñ° Expected events (doctor review, investigations)
‚ñ° Potential risks (falls, confusion, deterioration)
‚ñ° Family communication needs

---

### Example SBAR Handover Script

**"Mrs Smith, 78, Bed 12. Admitted yesterday with pneumonia.**

**Background:** Type 2 diabetes, hypertension, usually independent. NKDA. Normally lives alone, managing well. Admission via GP - 3 days of productive cough, fever, SOB.

**Assessment:** Currently stable. Last obs at 14:00 - T 37.2, BP 135/85, HR 88, RR 16, Sats 95% on 2L, NEWS 1. Completed IV co-amox this shift. Tolerating diet and fluids. Mobilizing to toilet with frame. No chest pain. Cough productive, green sputum.

**Recommendations:** Due for repeat bloods tomorrow morning. Consultant review expected on ward round 10am. Watch for deterioration - had spike to NEWS 5 on Day 1. Daughter wants update - number in notes. Physio seeing her tomorrow for chest physio."

## CHECKLIST 4: Clinical Emergency Quick Reference

**When seconds count - know what to do!**

### Cardiac Arrest (Adult)

**Immediate Actions:**
- [ ] SHOUT for help
- [ ] Press crash button/call 2222
- [ ] Start CPR (30:2 ratio)
- [ ] Get defibrillator
- [ ] Note time of arrest

**CPR Technique:**
- Hand position: Center of chest
- Depth: 5-6cm
- Rate: 100-120/minute
- Minimize interruptions
- Switch every 2 minutes

**When Defib Arrives:**
- [ ] Continue CPR while attaching pads
- [ ] Clear area for shock
- [ ] Follow defib prompts
- [ ] Continue until team arrives

---

### Sepsis Recognition (Sepsis Six)

**Give within 1 hour:**
1. [ ] Oxygen (target 94-98%)
2. [ ] IV fluids (500ml bolus)
3. [ ] IV antibiotics (broad spectrum)

**Take within 1 hour:**
4. [ ] Blood cultures
5. [ ] Lactate
6. [ ] Urine output monitoring

**Red Flags:**
- NEWS ‚â•5
- Systolic BP <90
- HR >130
- RR >25
- Needs oxygen for Sats >92%
- Non-blanching rash
- Recent chemo
- Altered mental state

---

### Anaphylaxis

**Immediate Actions:**
- [ ] Remove trigger if possible
- [ ] Call for help
- [ ] Lie patient flat (legs elevated if BP low)
- [ ] Give IM adrenaline 500mcg (0.5ml of 1:1000)
- [ ] High-flow oxygen
- [ ] IV access √ó 2
- [ ] IV fluids

**Repeat adrenaline after 5 mins if no improvement**

**Signs:**
- Airway swelling
- Wheeze/stridor
- Hypotension
- Urticaria/rash
- Angioedema

---

### Choking (Adult)

**Mild Obstruction (can cough):**
- Encourage coughing
- Stay with patient
- Monitor for deterioration

**Severe Obstruction (cannot cough/speak):**
- [ ] 5 back blows (heel of hand, between shoulder blades)
- [ ] 5 abdominal thrusts (Heimlich)
- [ ] Repeat cycle
- [ ] Call for help if not resolving
- [ ] Prepare for arrest

---

### Hypoglycemia

**BM <4.0 mmol/L + symptoms:**

**If conscious and can swallow:**
- [ ] 15-20g quick-acting carb (juice, glucose tablets, Lucozade)
- [ ] Recheck BM after 10-15 mins
- [ ] If still <4, repeat
- [ ] Once >4, give long-acting carb (sandwich, biscuits)

**If unconscious/cannot swallow:**
- [ ] Call for help
- [ ] IV access
- [ ] 100ml 20% glucose IV OR
- [ ] 1mg glucagon IM
- [ ] Protect airway
- [ ] Recheck BM 10 mins

---

### Falls

**Immediate:**
- [ ] Don't move patient immediately
- [ ] Assess for injury
- [ ] Check consciousness
- [ ] Check for head injury
- [ ] Vital signs
- [ ] Pain assessment

**If head injury:**
- [ ] Keep still
- [ ] C-spine precautions
- [ ] Medical review
- [ ] Neuro obs

**After fall:**
- [ ] Incident report
- [ ] Falls risk assessment
- [ ] Care plan update
- [ ] Family notification

## CHECKLIST 5: IV Cannulation Success - Every Time

**Increase your first-time success rate!**

### Pre-Procedure

**Preparation:**
- [ ] Check prescription/indication
- [ ] Explain to patient
- [ ] Gain consent
- [ ] Check allergies (if using LA)
- [ ] Gather equipment
- [ ] Hand hygiene
- [ ] Apron + gloves

**Equipment Needed:**
- [ ] Cannula (appropriate size)
- [ ] Tourniquet
- [ ] Alcohol wipe
- [ ] Gauze
- [ ] Tape/dressing
- [ ] Sharps bin nearby
- [ ] Flush (10ml saline)
- [ ] Extension set (if using)

---

### Vein Selection - Make it Easy!

**Look for:**
‚úÖ Springy, bouncy veins
‚úÖ Straight veins (not tortuous)
‚úÖ Larger veins (easier targets)
‚úÖ Non-dominant arm if possible
‚úÖ Away from joints
‚úÖ Away from previous attempts

**Avoid:**
‚ùå Bruised areas
‚ùå Over joints (flexes = tissue)
‚ùå Thrombosed veins (hard, cord-like)
‚ùå Side of mastectomy
‚ùå AV fistula arm
‚ùå Infected/broken skin

**Vein Tricks:**
- Warm compress 2 mins before
- Let arm hang down
- Tap gently
- Clench fist
- Hydrate patient if possible

---

### Size Guide

**Choose wisely:**
- **22G (blue):** Elderly, fragile veins, paeds
- **20G (pink):** Standard fluids, medications
- **18G (green):** Blood transfusion, rapid fluids
- **16G (gray):** Resus, major hemorrhage

**Rule:** Smallest size that meets clinical need

---

### Insertion Technique

**Step-by-Step:**
1. [ ] Apply tourniquet (not too tight!)
2. [ ] Identify vein
3. [ ] Clean with alcohol wipe (30 secs, allow to dry)
4. [ ] Anchor vein (pull skin taut)
5. [ ] Insert at 10-30¬∞ angle
6. [ ] Watch for flashback
7. [ ] Advance cannula slightly
8. [ ] Lower angle, advance cannula off needle
9. [ ] Release tourniquet
10. [ ] Press above cannula tip
11. [ ] Remove needle into sharps bin
12. [ ] Attach flush/extension
13. [ ] Flush to check patency
14. [ ] Secure with dressing
15. [ ] Document (date, time, site, size, your signature)

---

### Troubleshooting

**No flashback?**
- Too shallow - redirect slightly
- Through the vein - pull back slowly
- Wrong vein - try another

**Flashback but can't advance?**
- Valve blocking - try gentle flush
- Pulling on skin - release slightly
- Through vein - remove and try again

**Won't flush?**
- Tissue - remove, failed attempt
- Against valve - try gentle flush
- Blocked - remove, failed attempt

---

### Post-Procedure

- [ ] Check cannula flushing
- [ ] Check no swelling/extravasation
- [ ] Secure properly
- [ ] Date label on dressing
- [ ] Document in notes
- [ ] Dispose of sharps safely
- [ ] Remove gloves + hand hygiene

---

### Documentation

**Record:**
- Date + time
- Site (e.g., "left forearm ACF")
- Size + brand
- Indication
- Number of attempts
- Your signature + PIN
- VIP score if policy requires

---

- ‚ùå 2 failed attempts (ask colleague)

‚úÖ Practice on easier patients first
